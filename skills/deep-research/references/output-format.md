# Output Format Reference

## Report Template

Every research report follows this structure:

```markdown
# Research: [Topic]
Date: YYYY-MM-DD | Depth: Quick|Standard|Deep
Sources analyzed: N

## Executive Summary
[2-3 sentences: what was asked, what was found, bottom line]

## Key Findings
1. **[Finding statement]** — Source: [title](url)
2. **[Finding statement]** — Source: [title](url)
...

## Detailed Analysis
### [Subtopic 1]
[Analysis with supporting evidence]

### [Subtopic 2]
[Analysis with supporting evidence]

## Contradictions & Caveats
- [Source A says X, Source B says Y — likely because...]
- [Limitation: only English-language sources consulted]

## Sources
| # | Title | URL | Date |
|---|-------|-----|------|
| 1 | Article Title | https://... | 2026-01-15 |
```

## Comparison Tables

For competitive analysis or technology comparisons:

| Feature | Option A | Option B | Option C |
|---------|----------|----------|----------|
| Speed | Fast | Medium | Slow |
| Cost | $10/mo | Free | $50/mo |

## Timeline Format

For trend/historical analysis:
- **2024-Q1**: Event description — Source
- **2024-Q2**: Event description — Source

## Quality Checklist

Before finalizing:
- [ ] At least 3 independent sources
- [ ] Dates on all sources
- [ ] Contradictions noted (not hidden)
- [ ] Executive summary is honest about uncertainty
- [ ] No unverified claims presented as fact
